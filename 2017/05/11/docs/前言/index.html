这是一个很久前就想开始的开始，也是一个希望多年后仍不会结束的进行中。。。
这么多年，想说的话有很多，每一次灵感跃出，却没有马上要记录下来的意识，总之，错过了很多稍纵即逝的瞬间。等真的想写点东西的时候，却不知道从何开始了。
人都是感性动物，某一时刻的心情，可能只有在相似场景下才能复现。
徐静蕾在“朗读者”节目中被问及为什么不再拍文艺片的时候，回答说：“如果你不痛苦，怎么拍片子？”。
北岛也说过，作家就应该是孤独的。
对很多人而言，写作的创意，来源于生活，更来源于集矛盾与分裂为一体的自我演绎。回头想想，那些文思泉涌的瞬间，也大多都是内心饱受折磨或者思想受到冲撞的时刻。
我不想“痛苦”，所以，很多时候，我沉溺于丢三落四大大咧咧的状态中根本不想有所改变，更觉得不需要什么鬼灵感，不需要去记住对我不重要的任何事情，反正东西不会丢，我也不会丢。
可是却忽略了自己本就是一个心思细腻，怀旧恋旧的人，可能记不起数字和路牌，但潜意识里一定都记得发生过的事、说过的话和当时的场景。
即便是记得，由于人脑记忆的新陈代谢，那些想要留下的碎片，也终究会变得模糊。
在某段时间，我曾经迫切的想要失忆，久而久之终于变成了一个烂记性的人，可是现在，我更迫切地想把时间都定格。
如果有一天，我和奶奶一样，变成了阿尔茨海默病患者，只要我还认识字还能看得清，我都想翻翻那泛黄而略显潦草的一生，当然，前提是我能活到老年。
我知道，有个酷酷的老太太在未来里等我，我答应送她一个“岁月宝盒”的，嗯，就这样。

       新篇
